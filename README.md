# 江斌的个人简历网站

AI 应用工程师 / Python 后端开发工程师的个人作品集网站。

## 技术栈

- **框架**: Astro 5.x
- **样式**: Tailwind CSS
- **搜索**: Pagefind
- **部署**: Vercel

## 功能特性

- SEO 优化（sitemap、Open Graph、Twitter Card）
- RSS 订阅
- 暗色模式（跟随系统 + 手动切换）
- 全站搜索（Ctrl/Cmd+K）
- Vercel Analytics

## 开发命令

```bash
npm install     # 安装依赖
npm run dev     # 启动开发服务器
npm run build   # 构建生产版本
npm run preview # 预览构建结果
```

## 部署

项目自动部署到 Vercel，每次推送到 master 分支会自动触发部署。
